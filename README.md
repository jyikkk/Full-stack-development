# 🌱 Full-Stack Development (FSD) — Learning Journey  
**Singapore Polytechnic | Year 2, Semester 2**

This repository captures my hands-on learning journey throughout the Full-Stack Development (FSD) module at Singapore Polytechnic. It includes in-class exercises, practice projects, and experiments as I build up my skills from frontend fundamentals to backend development.

---

## 🚀 What I Learned

### **1. Frontend Development**
- HTML5 structure & semantics  
- CSS layouts (Flexbox, Grid)  
- Responsive design  
- Basic UI/UX principles  
- JavaScript fundamentals (DOM manipulation, events, functions)  

### **2. Backend Development**
- Node.js & Express.js basics  
- HTTP request handling  
- RESTful API concepts  
- Routing & middleware  

### **3. Databases**
- Introduction to SQL / NoSQL  
- CRUD operations  
- Connecting backend to databases  

### **4. Full-Stack Skills**
- Building small end-to-end applications  
- Integrating frontend with backend endpoints  
- Working with JSON and API responses  
- Deploying simple apps (if applicable)

This repo contains different **practice folders** and **assignments** completed throughout the semester, such as:

- 🎯 Small JavaScript exercises

The goal of this repo is to serve as a living journal of my progress as I grow into a full-stack developer.

---

## 🧠 Key Takeaways

- Full-stack development is a combination of frontend + backend thinking.
- Understanding core web concepts (HTTP, routes, DOM, JSON) is essential.
- Building projects improves learning far more than memorizing theory.
- Debugging is a critical skill I’m still improving every day.
